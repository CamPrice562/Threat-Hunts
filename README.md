# 🔍 Threat Hunt — The Buyer (Akira Ransomware)

**Platform:** SancLogic Cyber Range | **Difficulty:** Advanced | **Date:** March 2026

---

## Synopsis

A ransomware affiliate returned to the environment of Ashford Sterling Recruitment using a backdoor pre-staged during a prior attack. They deployed Akira ransomware, encrypted files across two servers, stole sensitive data, and demanded £65,000. Starting from a single Defender alert about Windows Defender being disabled, the full attack chain was reconstructed across 40 questions and 12 sections — covering initial access, credential theft, lateral movement, exfiltration, ransomware deployment, and anti-forensics. All 40 questions solved. This was my first threat hunt.
